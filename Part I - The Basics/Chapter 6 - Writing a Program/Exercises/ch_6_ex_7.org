* A simple grammar for bitwise logical expressions
  Timestamp: 18 Set 2018
  Chapter 6, Exercise 7
  + Expression
    - Unary
    - Expression & Unary		# and
    - Expression ^ Unary		# exclusive or, a.k.a xor
    - Expression | Unary		# or
  + Unary
    - Primary
    - ! Primary				# not
    - ~ Primary				# complement
  + Primary
    - Number
    - "(" Expression ")"		# grouping
  + Number
    - Literal integer
